This program is designed to carry out compass error and gyro error calculations for you.

Some manual calculation will still need to be carried out to determine LHA and Declination of the observed heavenly body.

Select one of the tabs above to get started, fill in the details and press 'Calculate'.

Disclaimer:
This program has been rigorously tested but it is impossible to test for every situation.
Computers use a lot more decimal places when calculating.
For example, my test showed that this program was using 17 decimal places in it's calculations.
As a result of this, you may see a slight discrepancy in the results obtained due to rounding.
